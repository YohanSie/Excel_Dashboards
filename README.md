# 📊 Dashboard de Vendas no Excel

## 📌 Sobre o Projeto
Este projeto tem como objetivo a criação de um dashboard sobre assinaturas do Xbox Game Pass.

A proposta é transformar dados brutos em informações visuais claras e úteis, permitindo uma análise eficiente do desempenho de vendas e apoiando a tomada de decisões baseada em dados.

---

## 🎯 Objetivos
- Organizar dados de vendas de forma estruturada  
- Criar visualizações claras e intuitivas  
- Facilitar a análise de desempenho  
- Apoiar a tomada de decisão com base em dados  

---

## 🛠️ Ferramentas Utilizadas
- Microsoft Excel  
  - Tabelas dinâmicas  
  - Gráficos  
  - Segmentações de dados (Slicers)  
  - Fórmulas
